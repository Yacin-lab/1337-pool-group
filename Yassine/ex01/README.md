# ft_print_alphabet

## Description
Write a function that prints the lowercase alphabet from `a` to `z` in ascending order.

The function must print the characters in a single line without spaces or separators.


Allowed function:  `write()`

---
## Bugs

**Wrong ASCII range**


> starting from 'A' instead of 'a'


**Incorrect loop condition**


> while (c < 'z')


> This skips printing `z`.

**Forgetting to increment**


> infinite loop


**Wrong write size**


> write(1, "\n", 2)


> Correct size:  write(1, "\n", 1)


**Printing newline accidentally**

> The exercise does not require a new line.
