# 1337 Piscine Group Repository

## 🎯​ Purpose

Shared repository to compare solutions for exercises from the 1337 Piscine.
Each member solves exercises individually and pushes their work for review and discussion.

## 📋​ Rules

* Solve exercises **alone first**.
* Push your solution **after finishing**.
* Do **not copy solutions** before solving.
* Use the repo to **compare logic and discuss bugs**.

## Repository Structure

```
1337-piscine-group
│
└──  person
    ├── C00
    └── C01

```

## ⚠️​ Contribution Rules

* Work only inside **your own folder**.
* Do not modify other members' folders.
* Commit messages should be clear.

Example:

```
git commit -m "C00 ex03 ft_print_numbers"
```

## 💬​ Discussion

Solutions and bugs are discussed on Discord after everyone pushes their work.

Workflow:

solve → push → compare → discuss → improve
